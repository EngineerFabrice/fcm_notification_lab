## 📱 Project Overview

This project demonstrates the integration of **Firebase Cloud Messaging (FCM)** into a Flutter application for sending and receiving push notifications.

### Features Implemented
- ✅ Request notification permissions (Android 13+)
- ✅ Retrieve and display FCM device token
- ✅ Receive push notifications from Firebase Console
- ✅ Display received messages in app UI
- ✅ Popup notifications when app is in foreground
- ✅ System tray notifications when app is in background
